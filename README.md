# Truck Parts Cloud App V2

Domains:
1. Vehicle disassembly / part-out inventory
2. Misc / vendor / overstock inventory

Setup:
1. Run `schema_v2.sql` in Supabase SQL Editor.
2. Copy files into your GitHub/VS Code project.
3. Add Streamlit secrets:
   SUPABASE_URL
   SUPABASE_KEY
4. Push to GitHub and redeploy Streamlit.
